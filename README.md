# 🤖 Multiverse Chatbot

A modern AI chatbot built with **Streamlit** and powered by the **Groq API**. Chat with multiple unique personalities in real time using fast LLM inference and a clean chat interface.

##  Features

- 🤖 Multiple AI personalities

## Installation

Clone the repository:

```bash
git clone https://github.com/shubodeep-tech/Multiverse-of-Chatbots.git
cd Multiverse-of-Chatbots
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install streamlit groq python-dotenv
```

---

##  Run the App

```bash
streamlit run app.py
```

The application will open at:

```
http://localhost:8501
```

---

## Groq API Key

Get your free API key from:

https://console.groq.com/keys

Enter your API key in the sidebar after launching the application.
